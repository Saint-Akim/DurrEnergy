## üìà Reading the Charts

### 1. Statistical Comparison Analysis

This comprehensive chart shows six key views:

#### Daily Generation with Confidence Intervals
- **Red line**: Legacy system (4 inverters)
- **Green line**: New system (3 inverters)  
- **Shaded area**: 95% confidence range
- **Orange dashed line**: System upgrade date (Nov 2025)

**How to read**: Look for the green line consistently above the red line, with minimal overlap in shaded confidence areas.

#### Industry Benchmark Comparison
- **Bars show**: Your system vs SA averages
- **Colors**: Red (legacy), Green (new), Gray (SA average), Orange (SA excellent)
- **Target**: New system bar should exceed SA average

#### Weather-Normalized Performance  
- **Purpose**: Shows true equipment performance independent of weather
- **Y-axis**: Performance index (100 = baseline)
- **Trend**: Should show clear upward improvement

### 2. New System Performance Analysis

#### Daily Capacity Factor
- **Y-axis**: Percentage of theoretical maximum output
- **Target line**: Industry benchmark (usually 25% for SA)
- **Good range**: Above the benchmark line consistently

#### Generation Distribution
- **Histogram**: Shows how often you achieve different generation levels
- **Peak**: Should be at a high generation value
- **Shape**: Narrow peak indicates consistent performance

## üèÜ Industry Benchmarking Explained

### South African Solar Standards

Your system is compared against these industry benchmarks:

| Performance Level | Capacity Factor | What it Means |
|-------------------|-----------------|---------------|
| **Excellent** | >28% | Top 10% of SA installations |
| **Good** | 22-28% | Above national average |
| **Average** | 18-22% | National average |
| **Poor** | <18% | Below average, needs attention |

#### Capacity Factor Calculation
```
Capacity Factor = (Actual Generation) / (Theoretical Maximum) √ó 100%
```

**Example**: If your 30kW system generates an average of 6.6kW continuously over a day:
```
Capacity Factor = 6.6kW / 30kW √ó 100% = 22%
```

### What Affects Your Ranking?
- **Geographic location** (solar resource availability)
- **System design** (panel orientation, tilt, shading)
- **Equipment quality** (inverter and panel efficiency)  
- **Maintenance** (cleaning, repairs, monitoring)

## üõ†Ô∏è Troubleshooting Common Issues

### Problem: "No solar data available"

**Possible Causes**:
- Data files not uploaded correctly
- Selected date range has no data  
- File format issues

**Solutions**:
1. Check that both CSV files are in the application folder
2. Verify date range includes actual generation periods
3. Contact IT support if files appear corrupted

### Problem: "Data quality is poor"

**Possible Causes**:
- Sensor malfunction  
- Network connectivity issues
- Database corruption

**Solutions**:
1. Check physical sensor connections
2. Verify network connectivity to monitoring system
3. Review data for obvious gaps or errors

### Problem: "Limited data for statistical confidence"

**Possible Causes**:
- Recently installed system  
- Large gaps in historical data
- Seasonal data only

**Solutions**:
1. Wait for more data to accumulate (recommended: 30+ days)
2. Use basic analysis mode temporarily  
3. Focus on recent performance trends

### Problem: Performance alerts showing

**Immediate Actions**:
1. **Visual Inspection**: Check for obvious shading, soiling, damage
2. **Inverter Status**: Look at inverter display panels for error codes  
3. **Environmental Factors**: Consider weather, seasonal changes
4. **Professional Assessment**: Contact technician if issues persist

## üìö Appendix

### Glossary of Terms

| Term | Definition |
|------|------------|
| **Capacity Factor** | Percentage of theoretical maximum power output actually achieved |
| **Clipping** | Power loss when panels generate more than inverters can handle |
| **Confidence Interval** | Statistical range showing uncertainty in performance estimates |
| **kWh** | Kilowatt-hours, unit of energy (how much electricity was generated) |
| **kW** | Kilowatts, unit of power (rate of electricity generation at a moment) |
| **ROI** | Return on Investment, time for savings to equal upgrade cost |
| **Seasonal Adjustment** | Mathematical correction for weather/daylight variations |

### Units and Conversions

| Metric | Unit | Typical Range |
|--------|------|---------------|
| Daily Generation | kWh | 50-200 kWh/day |
| Power Output | kW | 5-30 kW |
| Capacity Factor | % | 15-30% |
| Annual Savings | Rands | R5,000-R25,000 |
